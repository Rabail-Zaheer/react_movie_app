## Fellowship Journey and Project Completion

During my 3-month fellowship program, I had the opportunity to dive into the world of frontend development. I started from the basics and gradually progressed to more advanced topics, building various projects along the way. Here's an overview of my journey:

1. **HTML and CSS Fundamentals**

   - Learned the core concepts of HTML and CSS.
   - Applied my knowledge to create a portfolio project. Repository: [Portfolio](https://github.com/Rabail-Zaheer/Portfolio).

2. **Bootstrap Projects**

   - Explored Bootstrap framework for responsive web development.
   - Developed two projects showcasing Bootstrap features. Repositories: [Bootstrap Project 1](https://github.com/Rabail-Zaheer/BootstrapProject) and [Bootstrap Project 2](https://github.com/Rabail-Zaheer/BootstrapProject2).

3. **Tailwind CSS**

   - Explored Tailwind CSS for utility-first styling.
   - Created a parallax landing page using Tailwind CSS. Repository: [Parallax Landing Page](https://github.com/Rabail-Zaheer/Parallax-landing-page).

4. **JavaScript Fundamentals**

   - Delved into JavaScript to enhance interactivity and functionality.
   - Developed a simple weather application using JavaScript. Repository: [Weather App](https://github.com/Rabail-Zaheer/WeatherApp).

5. **ReactJS Final Project**

   - Learned ReactJS and its ecosystem.
   - Completed a final project, a movie application, using ReactJS. Repository: [React Movie App](https://github.com/Rabail-Zaheer/react_movie_app).

6. **Future Learning Goals**
   - Looking forward to building more exciting projects in ReactJS to strengthen my foundation.
   - Planning to explore Next.js to stay updated with current trends and technologies.
